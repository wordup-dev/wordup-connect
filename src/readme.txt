This WordPress plugin connects your WordPress installation with your private plugin and theme registry on https://wordup.dev

== Description ==

Wordup-connect give you access to your private plugin and theme directory on https://wordup.dev

Use wordup-connect for:

* Downloading your themes or plugins from wordup.dev
* Providing the automatic update functionality of WordPress for your private wordup plugins and themes  

See [wordup.dev](https://wordup.dev) for more details.

== Installation ==

Upload the plugins zip to the `/wp-content/plugins/` directory of your WordPress website. 
Or download and activate the plugin directly through the 'Plugins' menu in WordPress. 

You will find 'Wordup connect' menu in your WordPress admin tools.

You can also have a look at [wordup.dev](https://wordup.dev) for more details.
